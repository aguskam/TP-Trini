##TP-trini
En este TP se realiza una aplicacion la cual sirve para tomar lista de varios cursos y varias materias
En este TP se usa HTML, JS, CSS, SQL y Node, este ultimo se usa para conectar la BBDD a la app
##Modo de uso
Primero se ejecuta la instalacion de las dependencias necesarias en node con el comando npm install mysql2 express cors
despues se conecta a la bbdd con el programa de mysql ya ejecutado con la bbdd ya cargada, se conecta al server con el comando
node server.js
Se descargan los archivos del repositorio y se ejecuta el html, luego ya esta la aplicacion para ejecutarla y ver su correcto funcionamiento
